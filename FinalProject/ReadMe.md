## AVL Tree by Jakobe McPhail & Sean McElwee
The AVL Tree is a tree that is all about balancing where the height between the subtrees for any node cannot be more than one

## How to run the program
1. Make sure to open all the files(avltreedr.cpp, avltree.cpp, & avltree.h)
2. Compile the program as g++ avltree.cpp avltreedr.cpp -o TreeTest
3. Run program as ./TreeTest

## What to do in the program
1. You are given 7 options(Insert, Delete, Search, Display, Height, Clear, & Exit)
2. Enter the option you want through the keyboard(0-6)
3. Insert: Insert the value you want, program will tell you that the value was inserted
4. Delete: Insert the value you wanted deleted, program will tell you that the value was deleted or was not found
5. Search: Insert the value you want to search in the tree, program will tell you whether or not the value is in the tree
6. Display: Returns what the current AVL Tree looks like and the Pre-order, In-order, and Post-order
7. Height: Returns the current height of the AVL tree
8. Clear: Clears everything
9. Exit: Closes the program
